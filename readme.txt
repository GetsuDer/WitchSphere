Сборка: cmake Cmakelists; make; ./Witch_Sphere

1) Мой код сохраняет результат в файл с правильным именем!
2) По умолчанию рендерится в 512x512, -w size даст любой размер
3) Рендерится < 1 минуты
4) Два источника света - над верхней гранью и правее камеры
5) Есть подставка (она на самом деле куб)
6) Лучи преломляются, закодено полное внутреннее отражение
7) Сцена внутри - образец минимализма, значение -
мини-площадка для битвы в мире Края.
